# Gamification Card Pools

> 记录“游戏化更新”的抽奖卡池、道具卡池和弱社交道具草案。本文档用于沉淀内容想法，后续可继续扩展为表格、配置文件或后台管理数据。

## 定位

卡池负责承接抽奖系统的具体内容。

主设计文档回答“系统为什么存在、怎么闭环”；本文档回答“用户能抽到什么、这些东西怎么用、哪些规则要提前压住”。

字段规范独立记录在 `docs/superpowers/specs/2026-04-25-gamification-content-schema.md`。

## 卡池原则

1. 不做强惩罚，不做羞辱式点名。
2. 弱社交道具以提醒、邀请、保护、播报为主。
3. 被点名成员可以忽略，不扣分、不影响个人收益。
4. 涉及健身的保护道具不能白给健身收益。
5. 稀有卡尽量提供期待感，不直接造成不可控经济膨胀。
6. 道具名称保留“搞笑摸鱼风”，规则描述必须清晰。

## 卡池分层

第一版建议拆成四类：

- `普通收益`：小额银子，让抽奖不空手。
- `实用道具`：任务刷新、保底、轻量增益。
- `稀有暴击`：双倍、暴击、赛季贡献增强。
- `弱社交道具`：提醒、邀请、团队播报、请假保护。

第一版概率模型建议采用“低银子回收 + 高道具体验”。

| 分层 | 概率 | 作用 |
| --- | ---: | --- |
| 普通银子 | 45% | 每抽都有机会拿少量银子，但不靠它回本 |
| 实用道具 | 27% | 任务换班、小暴击、请假保护、保底类 |
| 弱社交道具 | 18% | 点名喝水、出门溜达、团队小喇叭等 |
| 趣味收藏 | 6% | 今日称号、高光贴纸、周报彩蛋 |
| 稀有暴击 | 4% | 双倍牛马券、赛季贡献加倍券等 |

## 普通银子卡池

因为抽奖券价格暂定为 `40` 银子 / 张，抽奖直接返还的银子期望值必须明显低于 `40`，避免银子购券套利。

第一版普通银子卡池建议：

| 奖励 | 单抽概率 |
| --- | ---: |
| `+5 银子` | 15% |
| `+10 银子` | 12% |
| `+20 银子` | 10% |
| `+40 银子` | 5% |
| `+80 银子` | 2% |
| `+120 银子` | 1% |

这组银子奖励的单抽期望约为 `8.75` 银子，低于 `40` 银子的购券成本。

银子奖励的定位：

- 让抽奖不空手
- 提供偶尔抽到大额银子的惊喜
- 不承担主要回本能力
- 不替代道具和暴击的体验价值

## 抽奖规则

当前已确认的抽奖规则：

- 抽奖券可以长期累积，不会当天过期。
- 抽奖券进入背包，作为用户资产展示。
- 支持单抽和十连抽。
- 十连抽优先消耗已有抽奖券。
- 抽奖券价格暂定为 `40` 银子 / 张。
- 付费券每日最多购买 `3` 张。
- 抽奖券不足十张时，可以用银子补齐，但受每日购券上限限制。
- 十连抽必须至少已有 `7` 张券，最多用当日购券额度补齐 `3` 张。
- 单抽没有保底。
- 十连抽至少保底 `1` 个实用道具或以上。
- 十连抽保底不保证稀有暴击。
- 如果十连没有抽到 `实用道具 / 弱社交道具 / 稀有暴击`，则把其中一个最低银子奖替换为实用道具。
- 免费获得的券和银子购买的券进入同一个券余额，不单独区分来源。

## 背包规则

当前已确认：

- 抽奖券永久有效。
- 所有道具永久有效。
- 真实福利券永久有效。
- 道具不通过过期时间回收。
- 经济边界主要通过每日 / 每周使用次数、叠加限制和结算规则控制。

## 使用与叠加规则

当前已确认：

- 所有影响收益的道具都必须由用户手动使用。
- 暴击类道具不可叠加。
- 小暴击和强暴击不能同时生效。
- 健身请假券不能触发任何暴击。
- 弱社交道具可以同时持有，但每日使用次数受限。

## 今日生效规则

当前已确认：

- 收益类增益道具使用后绑定当天，以 Asia/Shanghai 日期为准。
- 如果当天尚未完成真实健身打卡，道具进入“今日待生效”状态。
- 用户当天完成真实健身打卡后，道具生效并结算额外收益。
- 如果当天已经完成真实健身打卡，使用道具后可以立即对当天打卡补结算。
- 道具只作用于当天真实健身打卡，不跨天自动延续到下一次健身。
- 如果当天最终没有真实健身打卡，收益类增益不生效，也不触发请假券。
- 收益类增益建议在真实结算成功时才扣减库存，避免用户提前使用后未健身导致误吞道具。

## 使用次数限制

当前建议：

| 道具类型 | 限制 |
| --- | --- |
| 强暴击类道具 | 每人每周最多使用 `1` 次 |
| 小暴击类道具 | 每人每天最多使用 `1` 次 |
| 任务换班券 | 每人每天最多使用 `1` 次 |
| 弱社交点名类道具 | 每人每天最多发起 `2` 次 |
| 团队广播类道具 | 每队每天最多触发 `1` 次 |
| 瑞幸咖啡券 | 系统内不限制持有数量，兑换需要管理员线下确认 |

## 弱社交邀请规则

当前已确认：

- 弱社交道具库存永久有效。
- 用户使用弱社交道具后，会生成一条社交邀请。
- 社交邀请当天有效，以 Asia/Shanghai 日期为准。
- 对方可以确认响应，也可以忽略。
- 未响应邀请过了当天自动失效。
- 第一版弱社交响应不发银子，只结算响应记录、展示素材和后续周报 / 动态素材。
- 只有对方确认响应后，才生成响应记录或展示记录。
- 弱社交道具使用后，会通过企业微信群机器人发送提醒消息。
- 企业微信只负责送达提醒，系统内仍需保留邀请状态、响应状态和结算状态。
- 不把所有普通点名都写入团队动态，只沉淀高价值或多人响应事件。

## 办公室互助卡池

这是一组弱社交道具，重点是制造团队互动，而不是强制别人完成任务。

| 道具 | 类型 | 稀有度 | 效果 | 约束 |
| --- | --- | --- | --- | --- |
| 点名喝水令 | 提醒 | 普通 | 点名一位成员喝水，对方确认后生成响应记录 | 对方可忽略，不扣分，v1 不发银子 |
| 出门溜达令 | 提醒 | 普通 | 邀请一位成员起身走一圈，对方确认后生成轻动态 | 不强制完成 |
| 全员起立令 | 团队 | 稀有 | 发起一次全队起身提醒，记录当天响应人数 | 每人每日最多响应一次，v1 不发银子 |
| 今日闲聊令 | 社交 | 普通 | 邀请一位成员完成“把事办黄”，双方完成后生成响应记录 | 不影响四维全成判定，v1 不发银子 |
| 红盘情报令 | 输入 | 普通 | 点名一位成员分享今天看到的新东西，完成后进入今日小摘要 | 分享内容可很短 |
| 工位补水广播 | 团队 | 普通 | 给全队发一条喝水提示，不点名 | v1 不发银子 |
| 午后续命令 | 团队 | 普通 | 下午时段提醒全队休息几分钟 | 仅下午可用 |
| 摸鱼合法证 | 趣味 | 普通 | 当天四维全成后生成一个搞笑称号或动态 | 只影响展示 |
| 今日免打扰牌 | 保护 | 稀有 | 当天不接受别人点名提醒 | 不影响自己完成任务 |
| 健身请假券 | 保护 | 稀有 | 当天无法健身时保护连续记录不断联，并冻结下一次真实健身奖励档位 | 不算健身，不发券，不给资产，不推进赛季 |
| 断签保护券 | 保护 | 稀有 | 后续接 streak 系统时保护一次连续记录 | 不补发任何奖励 |
| 任务换班券 | 任务 | 普通 | 把当天一个四维任务换成同维度另一张卡 | 每天使用次数受限 |
| 帮同事递水券 | 社交 | 普通 | 邀请一位成员完成补水，对方确认后双方都有轻量记录 | 和点名喝水令可二选一保留 |
| 同步摸鱼券 | 社交 | 稀有 | 邀请一位成员一起完成同一个四维任务 | 双方都需要确认 |
| 团队小喇叭 | 播报 | 稀有 | 把一句轻量播报发到团队动态 | 需要频率限制 |
| 今日高光贴纸 | 收藏 | 普通 | 给自己或别人贴一个当天称号，进入当天展示或周报彩蛋 | 只影响展示 |

## 实用道具卡池

实用道具用于提供低强度、常用、不会破坏经济的个人操作收益。

它们应该比弱社交更偏个人操作，比稀有暴击更克制。抽到后用户能立刻理解用途，但不会直接替代健身打卡。

| 道具 | 类型 | 效果 | 约束 |
| --- | --- | --- | --- |
| 任务换班券 | 任务 | 把当天一个四维任务换成同维度另一张卡 | 每天最多用 `1` 次 |
| 小暴击券 | 增益 | 当日真实健身打卡个人资产 `1.5x` | 不影响赛季贡献，不可叠加 |
| 补水加班费 | 增益 | 今天完成“喝白白”后额外获得少量银子 | 每天最多触发 `1` 次 |
| 站立补贴 | 增益 | 今天完成“把电充绿”后额外获得少量银子 | 每天最多触发 `1` 次 |
| 情绪报销单 | 增益 | 今天完成“把事办黄”后额外获得少量银子 | 每天最多触发 `1` 次 |
| 情报经费 | 增益 | 今天完成“把股看红”后额外获得少量银子 | 每天最多触发 `1` 次 |
| 保底升级券 | 抽奖 | 下一次单抽至少不低于实用道具 | 不影响十连，不保稀有 |
| 九折购券卡 | 折扣 | 今天买抽奖券享受一次 `9` 折 | 每天最多用 `1` 次 |
| 摸鱼日历贴 | 展示 | 给今天日历加一个趣味贴纸 | 只影响展示 |
| 小喇叭草稿 | 播报 | 生成一条团队动态草稿，需用户确认发布 | 每天最多发布 `1` 次 |

第一版建议优先保留：

- `任务换班券`
- `小暴击券`
- `保底升级券`
- `九折购券卡`
- `补水加班费`
- `站立补贴`

`小暴击券` 只影响当日真实健身打卡的个人资产，不影响赛季贡献。赛季贡献翻倍应留给稀有暴击层，避免普通道具过快推高赛季进度。

## 稀有暴击卡池

稀有暴击是抽奖系统的期待感来源，但必须严格限制叠加和使用频率。

第一版建议原则：

- 稀有暴击默认只作用于当日真实健身打卡。
- 稀有暴击不可叠加。
- 稀有暴击不因健身请假券触发。
- 同一用户每周最多使用 `1` 张强暴击类道具。
- 最强暴击不开放银子直购，主要通过抽奖获得。

| 道具 | 类型 | 效果 | 约束 |
| --- | --- | --- | --- |
| 双倍牛马券 | 强暴击 | 当日真实健身打卡个人资产 `2x`，赛季贡献 `2x` | 每人每周最多用 `1` 张，不可叠加 |
| 赛季冲刺券 | 赛季 | 当日真实健身打卡赛季贡献 `2x` | 不影响个人资产，不可叠加 |
| 银子暴富券 | 资产 | 当日真实健身打卡个人资产 `2x` | 不影响赛季贡献，不可叠加 |
| 全队加油券 | 团队 | 当天发起一次团队加油，当日真实健身成员额外获得小额银子 | 每队每日最多触发 `1` 次 |
| 超级保底券 | 抽奖 | 下一次十连至少出现 `1` 个稀有暴击或真实福利 | 极低概率，不开放购买 |
| 瑞幸咖啡券 | 真实福利 | 可找管理员兑换一杯瑞幸咖啡 | 线下兑换，不等同于系统咖啡打卡 |

`瑞幸咖啡券` 属于真实福利兑换，不直接改变系统内咖啡记录。用户兑换后，如果实际喝了咖啡，可以再按正常流程完成咖啡打卡。

- `点名喝水令`

- `红盘情报令`

- 团队播报

## 健身请假券规则

`健身请假券` 是保护类道具，不是补签奖励。

明确规则：

- 使用当天不创建真实健身打卡结果
- 当天不发放健身抽奖券
- 当天不获得个人资产
- 当天不推进赛季贡献
- 当天不触发任何双倍券或暴击券
- 当天连续记录不断联
- 下一次真实健身打卡沿用原本应获得的奖励档位

示例：

用户连续打卡已经来到一个本应获得 `40` 银子的档位。如果今天无法健身并使用健身请假券，则今天没有 `40` 银子，也没有赛季贡献；明天真实健身时，仍按 `40` 银子的档位发放。
