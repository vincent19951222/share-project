# 脱脂牛马 0.1.0 发布说明

这是「脱脂牛马」的第一个 MVP 版本。我们想先把最核心的体验跑通：让一个健身小队每天能在同一张看板上打卡、互相看见进度、共享提醒，并通过战报看到团队这一阶段坚持得怎么样。

这个版本已经支持团队协同打卡、今日打卡撤销、共享看板、团队通告、训练笔记和战报中心。大家可以用它记录每天是否完成训练，也可以在看板上发布约练、提醒、心得或临时公告。

我们也加入了轻量的激励系统。每次打卡都会获得个人资产「银子」，连续打卡会带来更高奖励；团队层面则有「牛马金库」和「赛季目标」，用来展示大家共同积累的训练贡献和阶段性进度。

战报中心会根据真实打卡数据生成本月表现概览，包括团队完成情况、活跃趋势和一些高光时刻。它的目标不是做复杂数据分析，而是帮助团队快速复盘：最近节奏稳不稳，哪些日子表现好，接下来要不要互相拉一把。

`0.1.0` 仍然是内部试用版本。我们会优先观察打卡是否顺手、看板是否真的能促进互动、银子和赛季目标是否有激励感，再决定后续要不要继续扩展团队管理、排行榜、资产玩法和更完整的长期目标系统。
