# 脱脂牛马 0.2.0 发布说明

`0.2.0` 是一次围绕「日常记录更完整」的小版本更新。除了原本的健身打卡，我们把团队里最真实、最容易被忽略的续命行为也纳入了看板：喝咖啡。

这个版本新增了「续命咖啡」打卡页面。大家可以记录当天喝了几杯咖啡，随时 +1 或撤回最新一杯；团队月历会同步展示每个人每天的咖啡杯数，让“今天谁又靠咖啡硬撑”这件事变得一目了然。咖啡打卡不参与银子和赛季奖励，它更像是团队里的轻量生活记录：不考核，只围观。

我们也新增了「牛马日历」页面，把健身打卡和咖啡打卡放进同一个月视图里。用户可以按月份查看自己的训练日、咖啡日和累计数据，快速回看这一阵子的生活节奏：哪几天认真动了，哪几天纯靠咖啡续命，哪些日子两件事都发生了。

战报中心也做了一次整理。原来的三张播报卡被替换为「咖啡能量站」，展示今日全队咖啡杯数、今日续命人数、本月累计、本周咖啡王和近 7 天咖啡因波形。它不是严肃的数据分析，而是一块带点吐槽感的团队状态小窗：看看大家今天是靠肌肉上班，还是靠咖啡开机。

在体验细节上，续命月历会自动把当前日期滚动到视野中心，减少横向找日期的麻烦；咖啡打卡后的日历、动态和战报也会保持同步。这个版本的重点不是增加复杂玩法，而是让团队的日常节奏被更自然地记录下来。

`0.2.0` 仍然是内部试用版本。接下来我们会继续观察咖啡记录和牛马日历是否真的有用：它们是会变成团队里的轻松话题，还是只是多一个按钮。如果反馈不错，后续可以继续扩展周/月复盘、个人节奏洞察，以及更有梗但不过度打扰的团队提醒。
